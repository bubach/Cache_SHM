§ releaseLocks()

static releaseLocks (   $internalPersistentId,
  $isAtShutdown 
)
static

Releases all locks in a storage.

This function is not meant to be used directly. It is implicitly called by the the destructor and as a shutdown function when the request ends. One of these calls ends up releasing any unreleased locks obtained during the request. A lock is also implicitly released as soon as there are no objects left in the current request using the same persistent ID.

Parameters
string$internalPersistentIdThe internal persistent ID, the locks of which are being released.
bool$isAtShutdownWhether the function was executed at shutdown.
Returns
void
