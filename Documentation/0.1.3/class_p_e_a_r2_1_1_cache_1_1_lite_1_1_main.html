setAutomaticCleaningFactor (   $automaticCleaningFactor)

Sets the level of automatic cleaning to use.

The automatic cleaning process destroys expired files when a new file is written.

  • 0 - no automatic cleaning is performed
  • 1 - cleaning is always performed
  • n - cleaning is performed randomly with a probability of 1 / n for each cache write.
Parameters
integer$automaticCleaningFactorthe level of automatic cleaning to use.
Returns
the current object, for fluent interface.

setReadControlType (   $readControlType)

Sets the read control hash type.

Must be one of:

  • md5 - best, but slowest
  • crc32 - slightly less safe, but faster (default)
  • strlen - fastest, but less likely to detect errors
Parameters
string$readControlTypethe type to set. Must be a valid type.
Returns
the current object, for fluent interface.
Exceptions
PEAR2 if the specified $readControlType is not a valid hash type.

setWriteControl (   $writeControl)

Sets whether or not write control is enabled.

When write control is enabled, the cache is read immediately after every write operation. Enabling write control will slow the cache down slightly only when saving cache values. Write control can detect some corrupt cache files.

Parameters
boolean$writeControlwhether or not write control is enabled.
Returns
the current object, for fluent interface.
